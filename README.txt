# Database_BestBuy

A MySQL database project simulating a BestBuy retail store system.
Built as a final project for a university database course.

## Tables

- Store, BrickAndMortar, Online
- Product, Electronics, Appliances, Toys, Furniture
- Customer, Member, Guest
- Vendors, Inventory, Discount
- Order, Order_Item, Order_Payment, Payment

## How to Run

1. Open MySQL
2. Run `schema.sql` to create the tables
3. Run any query files to test the database

## Notes

- Built with MySQL
- Interface was built using Python in Jupyter Labs
